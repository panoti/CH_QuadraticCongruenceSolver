# Quadratic Congruence Solver
----

Solving quadratic congruence equation on Z_p
```
a*x^2 + b*x + c = 0 (mod p)
```
by using Tonelli-Shanks algorithm

## Installation

1. Install **Ubuntu**
2. Install python version 3 (test with python 3.5.2):
  ```shell
  sudo apt-get install python3
  ```
3. `python3 main.py`
